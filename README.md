# AI 随口记

AI 随口记是一个手机端 AI 语音碎片整理 App 原型，目标是把用户随口说出的零散语音内容，自动整理成待办、提醒、购物清单、消息草稿、项目灵感和每日汇总。

## 核心流程

语音输入 → 文本转写 → MiMo API 理解 → 意图分类 → 多事项拆分 → 缺失信息追问 → 行动卡片生成 → 今日整理

## 当前状态

目前已完成移动端前端原型，包含首页、AI 拆解结果页、今日整理页、全部记录页和设置页。当前使用模拟数据展示交互流程，后续计划接入 Xiaomi MiMo API 完成真实语义理解和结构化输出。

## Demo 状态

当前仓库包含 AI 随口记的移动端前端原型代码，已完成首页、AI 拆解结果页、今日整理页、全部记录页、设置页和 Agent 工作流页。当前版本使用模拟数据展示语音转行动流程，后续计划接入 Xiaomi MiMo API，实现真实语义理解、意图分类、多事项拆分和结构化 JSON 输出。
